Descarga e instalación

Para descargar bootstrap se puede dirigir al sitio web y adaptarlo a su plantilla pre diseñada, copiando los archivos a su carpeta. O también puede descargar plantillas que ofrece bootstrap en la sección de temas. En este proyecto se descargó la plantilla "Business casual" cuyo paquete ya incluye los archivos de bootstrap. También cuenta con diseño adaptable.

Configuración

Configurar la plantilla lo hará modificando tanto el html como el css. No es recomendado tocar el archivo "bootstrap.css" puesto que esta es una librería. Se deberán hacer los cambios en el archivo css de la plantilla. En este caso es el "business-casual.css". Para crear nuevas clases deberá incluir el identificador en las etiquetas del archivo html, que luego van a ser llamadas en la hoja de estilo.
